**FeatureCard** — offering / value-prop card with icon tile, badge and action link.

```jsx
<FeatureCard icon="wand-sparkles" title="Landing Page Maker" badge="Available now" badgeTone="success" action="Start free">Upload a PDF and build a landing page in minutes.</FeatureCard>
```

accents: tomato, macaroni, muffin, rhino. Providing `action` makes the whole card a link.
