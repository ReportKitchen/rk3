**Button** — the primary action control; use for CTAs and form submits.

```jsx
<Button variant="primary" iconRight="arrow-right">Get started</Button>
<Button variant="secondary">See our work</Button>
<Button variant="accent" size="lg">Book a tasting</Button>
```

Variants: primary (tomato), secondary (rhino outline), accent (macaroni), ghost. Sizes sm/md/lg. `pill`, `full`, `href` to render as link. Icons are Lucide ids.
