Icons — Lucide vs. Special Lineal (Magnific)

Lucide (current)

  • Free & open source (ISC) — no license needed.
  • Loads from CDN or as a WordPress-friendly SVG set.
  • ~1,600 icons, one consistent ~2px rounded-stroke style.
  • Pairs cleanly with the rounded logo & headings.
  • Thinner coverage for niche info-design concepts.

Special Lineal — Magnific

  • ~288k icons — far deeper, great for Pattern Library specifics.
  • Proprietary: needs a Magnific/Freepik license + attribution.
  • No icon-font/CDN — download per-icon SVGs and self-host.
  • Distinct detailed line style; check it reads at small sizes.
  • More setup + governance to keep the set consistent.
Recommendation: keep Lucide as the default system set for UI and common concepts, and reach for Special Lineal selectively in the Pattern Library where you need a specific, expressive glyph.
