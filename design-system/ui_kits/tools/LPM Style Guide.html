Tools — Internal Style Guide

Shared UI vocabulary for Landing Page Maker and the rest of the RK tools family. Never ship a browser-default control: everything on this page has a styled equivalent. Copy the CSS, don't improvise.

Typography

Two families only. Bricolage Grotesque for display and headings, Mulish for everything else — body, labels, buttons, wayfinding. Monospace is not a design font: it appears only inside code blocks (export snippets).

Color

The tools live on RK's palette but with their own dosage: rhino-900 chrome, macaroni for selection and progress, tomato reserved for primary actions and destructive warnings. Paper, not white, behind every canvas.

Buttons

One tomato button per view, maximum. Rhino for secondary structure actions, outline/ghost for everything else.

Form controls

Real elements, styled — keyboard and screen-reader friendly. Focus is always a rhino border plus a macaroni-100 halo.
