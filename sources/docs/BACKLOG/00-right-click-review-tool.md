
# Right-Click Review Tool

Upgrading our current click-to-leave-feedback capability.

- The tool should be available from any tab on the front end (convert document, landing page, info design, future ones)
- What options are in its menu, or whether it functions at all, is determined by the tab.  So you can't try setting info patterns in the source, when you're on the landing page tab.


User can
1. select a run of text, including multiple paragraphs/elements
2. click an element

Then user Right-clicks to get a menu:

- create new assertion
- leave feedback
