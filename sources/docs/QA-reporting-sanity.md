
## body and aside headings should be semantic

Promote to a heading when it names the callout’s content:

- Key Finding
- What This Means
- Implementation Considerations
- A Note About Costs
- Community Voice

Do not necessarily promote when it is just a category badge:

- NOTE
- TIP
- CASE STUDY
- EXAMPLE
- FUN FACT

The reason: lots of generic “Note” / “Tip” headings can pollute the heading outline without helping navigation.
